# Bena Bazar Portfolio

This is a modern, professional portfolio website for Bena Bazar, built with React and Vite.

## Features
- **Premium Design**: Dark theme with Maroon and Gold accents matching the Bena Bazar brand.
- **Animations**: Smooth reveal effects using Framer Motion.
- **Responsive**: Works perfectly on mobile and desktop.
- **Bilingual**: Ready for Kurdish and English content.

## How to Run

1. Open a terminal in this folder.
2. Run `npm install` (if you haven't already).
3. Run `npm run dev` to start the local server.
4. Open the link shown (usually `http://localhost:5173`).

## Structure
- `src/components`: Contains all page sections (Navbar, Hero, Services, etc).
- `src/index.css`: Global styles and color variables.
- `public/`: Contains static assets like the logo.

## Customization
You can change the colors in `src/index.css` under the `:root` variables.
